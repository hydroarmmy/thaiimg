SVG เหมาะกับเว็บไซต์และงานออกแบบ

SVG (Scalable Vector Graphics) เป็นฟอร์แมตที่ ขยายได้ไม่เสียคุณภาพ เหมาะกับการใช้ในเว็บไซต์, โลโก้, ไอคอน thaiimg.com ช่วยห่อรูป raster ของคุณให้อยู่ในรูป SVG container พร้อมใช้งานใน HTML, CSS, หรือ design tools

SVG คืออะไร? ต่างจากรูปทั่วไปอย่างไร

SVG (Scalable Vector Graphics) เป็นฟอร์แมตที่ใช้ คณิตศาสตร์ ในการอธิบายรูปภาพ แทนการเก็บข้อมูลของแต่ละพิกเซลเหมือน JPG/PNG ทำให้ SVG ขยายได้ไม่จำกัดโดยไม่เสียคุณภาพ ไม่ว่าจะดูบนหน้าจอเล็กๆ ของมือถือ หรือป้ายโฆษณาขนาดยักษ์ก็ยังคมชัดเหมือนเดิม นอกจากนี้ SVG ยังเป็น ไฟล์ text-based สามารถเปิดด้วย text editor และแก้ไขด้วยโค้ดได้ จึงเหมาะมากกับการใช้ในเว็บไซต์และงานออกแบบ

การใช้ SVG container สำหรับรูป raster

thaiimg.com ใช้การห่อรูป raster (JPG/PNG/WEBP) ไว้ใน SVG container ทำให้ได้ไฟล์ .svg ที่ใช้งานได้ใน HTML, CSS หรือ design tools ต่างๆ แม้ว่ารูปข้างในจะยังเป็น raster อยู่ แต่การห่อใน SVG ก็มีประโยชน์ เช่น สามารถเพิ่ม overlay ข้อความหรือกราฟิก vector ลงไปบนรูปได้, ใช้กับ CSS animation ได้, หรือ embed ในไฟล์ document ได้สะดวกกว่า สำหรับการแปลงรูปถ่ายเป็น true vector จะต้องใช้กระบวนการ vectorize ซึ่งต้องการซอฟต์แวร์เฉพาะทาง

SVG ใช้กับอะไรได้บ้าง

SVG ใช้ได้กับ (1) เว็บไซต์ ทั้งใส่ใน HTML ตรงๆ หรือใช้เป็นพื้นหลัง CSS, (2) Design software เช่น Figma, Adobe Illustrator, Sketch, Inkscape, (3) Office tools เช่น PowerPoint, Word รุ่นใหม่ๆ, (4) Favicon ของเว็บไซต์ เพราะคมชัดทุกขนาด, (5) UI icons ในแอป mobile/desktop เพราะใช้สีจาก theme ได้ ข้อดีอีกอย่างคือ SVG รองรับ animation ผ่าน CSS และ JavaScript ทำให้สร้าง interactive graphics ได้

คำถามที่พบบ่อย

SVG ที่ได้เป็น vector หรือเปล่า?

ไม่ครับ เป็น raster image ที่ห่อใน SVG container การแปลงรูปถ่ายเป็น vector จริงๆ ต้องใช้โปรแกรมเฉพาะ (vectorize)

SVG ใช้ทำอะไรได้บ้าง?

ใช้ใน HTML/CSS เว็บไซต์, นำเข้า Figma/Illustrator, ใช้เป็น favicon, ไอคอน UI ได้หมด
