เทคโนโลยี

thaiimg ใช้เทคโนโลยี modern web เพื่อทำงานในเบราว์เซอร์:

  • Canvas API — บีบอัด ปรับขนาด ครอบตัด ฟิลเตอร์
  • jsPDF — สร้างไฟล์ PDF จากรูปภาพ
  • gif.js — สร้าง GIF animation
  • heic2any — แปลงไฟล์ HEIC จาก iPhone
  • browser-image-compression — บีบอัดอัจฉริยะ
  • @imgly/background-removal — ตัดพื้นหลังด้วย AI (ONNX model ทำงานในเครื่อง)
