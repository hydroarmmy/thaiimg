ความปลอดภัยและความเป็นส่วนตัว

เราออกแบบ thaiimg ให้ ปลอดภัยที่สุดเท่าที่จะเป็นไปได้ โดยใช้สถาปัตยกรรมที่ภาพของคุณ ไม่ถูกอัปโหลดไปไหนเลย

ภาพของคุณไม่ออกจากเครื่องคุณ
เครื่องมือทุกตัวบนเว็บนี้ทำงานโดยใช้ JavaScript ภายในเบราว์เซอร์ของคุณเอง — ไฟล์ภาพไม่เคยถูกส่งไปยังเซิร์ฟเวอร์ของเรา หรือเซิร์ฟเวอร์ของใครเลย

ทำไมเว็บอื่นถึงต้องลบไฟล์ใน 15-30 นาที?

เพราะเว็บส่วนใหญ่ อัปโหลดไฟล์ไป server เพื่อประมวลผล แล้วต้องสัญญาว่าจะลบทิ้ง ผู้ใช้ต้องเชื่อใจว่าเขาลบจริง

thaiimg ใช้วิธีต่างออกไป — ไม่ส่งไป server ตั้งแต่แรก ทุกอย่างทำใน browser โดยตรง เปิด DevTools ดู Network tab ได้เลยว่าไม่มีการอัปโหลดภาพ
